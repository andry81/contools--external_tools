1. DESCRIPTION
-------------------------------------------------------------------------------
External command line tools:
  * python-win32 (3.6.1)
  * collabnet-win32 (1.9.5-1, 1.8.17-1, 1.7.19.1)
  * cygwin-win32 (1.9.5-1, 1.8.17-1, 1.7.14-1)
  * sliksvn-win32 (1.9.5, 1.8.17, 1.7.22)
  * tortoisesvn-win32 (1.9.5.27581-1.9.5, 1.8.12.26645-1.8.14, 1.7.15.25753-1.7.18)
  * visualsvn-win32 (1.9.5)
  * win32svn (1.8.17, 1.7.22)

The latest version is here: https://sf.net/p/contools

WARNING:
  Use the SVN access to find out new functionality and bug fixes.
  See the REPOSITORIES section.

-------------------------------------------------------------------------------
2. DEPENDENCIES
-------------------------------------------------------------------------------
External dependencies:
  * Windows UCRT runtime for the app-local distribution (from the Windows 10
    SDK).
    Might require to run the python and other tools. If so then copy them into
    the root of the application executable.
    See details here: https://blogs.msdn.microsoft.com/vcblog/2015/03/03/introducing-the-universal-crt/
